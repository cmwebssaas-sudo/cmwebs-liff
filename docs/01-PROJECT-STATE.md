# 專案現況

## 產品定位

CMWebs 智能租管是台灣長租房東管理系統，主要對象為：

- 自有房源房東
- 房東團隊
- 包租代管業者

核心生命週期：

```text
空房／建立房源
→ 建立房客與租約
→ LINE 綁定
→ 入住報到
→ 帳單與抄表
→ 付款回報與核准
→ 催繳
→ 訊息／報修
→ 續約或退租
→ 押金結算
→ 房間恢復空房
```

## 版本目標

- **V2**：內部自有房源可正式上線使用。
- **V3**：可對外正式上線並收費的 SaaS。
- **V4**：AI 房源內容、影片、社群與自媒體自動推廣。

## 目前完成度

完成度是產品規劃估計，不代表自動測試覆蓋率：

| 目標 | 估計完成度 | 說明 |
|---|---:|---|
| V2 內部房源上線 | 68–72% | 核心流程大多存在，仍缺整併、報表、完整報修、退租及穩定性 |
| V3 對外收費 SaaS | 45–50% | 租管核心可沿用，商業化、資料庫、安全、訂閱與營運後台尚缺 |
| V4 AI 行銷 | 5–10% | 目前主要停留在長期產品規劃 |
