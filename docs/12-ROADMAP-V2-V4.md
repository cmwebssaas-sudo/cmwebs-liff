# 版本路線圖

## V2：內部自有房源正式上線

1. Production consolidation

### V2 完成定義

自有房源連續實際使用，主要流程不需回到人工 Excel／零散 LINE：

```text
建立房源
→ 房客與租約
→ 入住
→ 帳單
→ 收款
→ 催繳
→ 報修
→ 續約／退租
→ 押金結算
```
